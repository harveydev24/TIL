# 마크다운(Markdown) 정리

## 헤더

`# string` 

결과:

# string

string이 크게나타남. #의 개수를 통해 크기 조절 가능(1개 ~ 6개)

## 리스트

### 순서가 있는 리스트

```
1. string1
2. string2
3. string3
```

결과:

1. string1
2. string2
3. string3

순서가 없는 리스트

```
- string1
- string2
- string3
```

결과:

- string1
- string2
- string3

-, *, + 지원

### 인용 

```
> string
```

결과:

> string

### 링크 

```
[string](url)
```

결과:

[goolge](https:google.com)
